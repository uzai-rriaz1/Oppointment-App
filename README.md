# 🏥 Medify

Medify is a full stack doctor appointment booking platform built with the MERN stack.  
It allows patients to book appointments, doctors to manage schedules, and provides a modern dashboard experience for both users.

---

## 🚀 Features

### 👤 Authentication
- JWT Authentication
- Role-based access (Patient / Doctor / Admin)
- Protected Routes

### 🩺 Patient Features
- Browse doctors
- Book appointments
- View appointment status
- Responsive patient dashboard

### 👨‍⚕️ Doctor Features
- Manage availability slots
- Accept / Reject appointments
- View today's appointments
- Doctor dashboard

### ⚡ General
- Modern responsive UI
- Real-time appointment handling
- Secure backend APIs
- Optimized database structure

---
